Project->library inventory & borrowing system
Project Overview=
The library inventory & borrowing system is a python based CLI application designed to help libraries manage books,student borrowing records and inventory efficiently.This system replaces manual registers with a digital approach...

Project Structure=
library_manager/
│
├── library.py         # Main Python program
└── README.md          # Project documentation

key features=
1-Add new books with details such as book ID,title,author and copies.
2-Display all books in a tabular format.
3-Search books by Book ID or Title.
4-Borrow books.
5-Return books.
6-View all borrowed books using list.

Learning objectives=
1-Python data structures-list,dictionarie,tuples,sets.
2-Conditionals(if,elif,else)and loops(for,while).
3-Functions,recursions.
5-Data entry,record updating and reporting.
6-Modular program design and menu-driven CLI.
7-String formatting.

how to use=
Run the program:
LIBRARY_MANAGER
On running the program CHOOSE B/W-
1-Add Book
2-View Books
3-Search Book
4-Borrow Book
5-Return Book
6-Exit

Enter the corresponding number to perform an operation.
Follow prompts to enter book or student details.
Borrowing and returning books will automatically update the inventory.
The program continues running until you choose Exit.
